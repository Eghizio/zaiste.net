Flutter in Practice

{% markdown %} **Flutter in Practice** is a free programming course that teaches how to write a mobile application using [Flutter](https://flutter.io/) framework and [Dart](https://www.dartlang.org/) programming language. Instead of teaching concepts and ideas in both Flutter and Dart, this course showcases **a process of building an actual, mobile application**. The goal for this tutorial is to create an **email client** that exclusively supports JMAP protocol. {% endmarkdown %}
