* How To: Find the Number of CPU cores

#+BEGIN_SRC bash
cat /proc/cpuinfo | grep -c processor
#+END_SRC

#+BEGIN_SRC bash
cat /proc/cpuinfo | grep 'core id'

core id         : 1
#+END_SRC
